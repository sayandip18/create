# Create

Create is a Codepen clone. The user can type HTML, CSS and JS code in the respective editors
to see the output in the terminal below.

![demo](chrome-capture.gif)


## To run

```
git clone git@github.com:sayandip18/create.git
cd create
npm install
npm start
```

## Technologies used

* React
* Codemirror
